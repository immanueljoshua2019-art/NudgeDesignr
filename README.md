# NudgeDesignr
NudgeDesigner is an AI-powered behavioural audit tool that analyzes campaigns, policies, and communications to diagnose why they fail to change behaviour, and redesigns them using evidence-based nudge science.

Built on a framework of 20 behavioural levers, 54 published RCTs, and the EAST framework, it scores any message on a Behavioural Effectiveness Index, identifies which psychological mechanisms are present or missing, generates a redesigned version, and benchmarks it against real-world case studies.

Designed specifically for African policy and advertising contexts, where Western behavioral tools consistently misfire.
